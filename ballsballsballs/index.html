2d collision of 'spherical' hard bodies


W-A-S-D to accelerate all
'G': toggle gravity + drag
'C' to spawn more balls
'X' to toggle SIZE of future balls
'K' to toggle clearCanvas();
'M' to mute the geiger sound ._.
'P': pause/unpause || 'R': [RESET]

source code on github

You can turn off gravity (G), turn off clearCanvas (G), and observe brownian motion on the larger balls. Keep in mind that gravity increases the number of collisions thus drastically decreasing performance, because at the time I wrote this app I was still a learner and honestly, the code is worse than pubic hair.

You can also just Reset (R), toggle size of balls (X), spawn a bunch of tiny balls by holding down (C), make sure that the collision sound is on (M), and get mildly surprised by how seldom the collisions actually occur for a screen full of tiny balls.
